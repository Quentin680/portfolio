    Project summary for the portfolio

    Context & objective

    The objective of this SAE was to create a personal website in the form of an online CV. I had to design a main page presenting my profile, training, diplomas, skills and personal information. The site also had to include several secondary pages to detail certain parts of the CV, with links and anchors to make navigation easier. The goal was to put the basics of HTML5 and CSS3 into practice in order to produce a clear, structured, attractive and responsive website.

    Process & methodology

    The project began with a study of the requirements document in order to identify the different information to present. I then designed the structure of the site by creating a main page and several additional pages. Once the structure was completed, I developed the different pages in HTML and used CSS to style the entire site and ensure a consistent presentation. Finally, I added links and anchors to make it easy to navigate between the different sections before carrying out the necessary tests and corrections.

    Feedback

    What I learned

    This SAE allowed me to discover the basics of web development and better understand the structure of an HTML page as well as the use of CSS stylesheets. I learned to design a website made up of several pages, to use CSS Grid and Flexbox to organise content and to improve the presentation of a web interface. This project also helped me develop rigour in writing code and the ability to correct errors encountered.

    What I would do differently

    With hindsight, I would spend more time on the website design phase before starting development. I would create a more detailed mock-up in order to have an overall view of the organisation of the different pages and their appearance. I would also pay more attention to the visual aspect and usability of the site in order to offer smoother navigation and a more professional presentation.
